주요 특징 (Key Features)

통합 음성 오케스트레이션

Server-Side Trigger: 음성 인식 완료 시 브라우저 거치지 않고 서버에서 즉시 에이전트 작업 실행. OpenAI Realtime API 기반 실시간 STT 스트리밍.

엔터프라이즈급 신뢰성

Redis BRPOPLPUSH 패턴으로 At-Least-Once 작업 전달 보장. CAS(Compare-And-Set)로 분산 환경 상태 정합성 유지. 소켓 타임아웃 20s 최적화.

지능형 계획 및 관리

prompts.yml 중앙 프롬프트 관리. HITL(Human-In-The-Loop) 정책 게이트. Circuit Breaker로 장애 에이전트 자동 차단.

시작하기 (Quick Start)

1. 가상환경 및 의존성 구축

모든 서비스는 프로젝트 루트의 통합 가상환경(.venv)에서 실행. Supervisor 의존성을 기준으로 환경 구축.

# 가상환경 생성
python3 -m venv .venv

# 가상환경 활성화
source .venv/bin/activate

# 슈퍼바이저 기준 의존성 설치
pip install -r src/supervisor-agent/requirements.txt

2. 서비스 기동

./start_all.sh

3. 기능 검증

python verify_session_id_e2e.py
